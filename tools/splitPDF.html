Split PDF

Split a PDF into multiple files in one go.

Example: with 2, a 10-page PDF becomes five PDFs of 2 pages each.

Recent history

What is PDF Splitter?

PDF Splitter extracts specific pages or divides large PDF documents into multiple smaller files. Choose between splitting every page into separate files or grouping pages into fixed-size chunks. All split files are delivered in a convenient ZIP archive.

How to Use This Tool

  1. Upload PDF: Select your PDF file to split.
  2. Choose Split Mode: Select "One page per PDF" or "Fixed number of pages per file".
  3. Set Parameters: For fixed mode, specify pages per file.
  4. Split: Click "Split PDF" to process and download all files.

Frequently Asked Questions (FAQ)

What split criteria should I use?

Use "One page per PDF" for individual page extraction, or "Fixed pages per file" to create documents of specific lengths.

How many output files will I get?

For "one page per PDF", you'll get as many files as pages. For fixed mode, it's total pages divided by pages per file (rounded up).

Can I extract specific page ranges?

The current tool splits the entire document. For specific ranges, consider using the fixed pages mode with appropriate sizing.
