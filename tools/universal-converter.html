Frequently Asked Questions

What is the difference between Data Storage and Network Speed?

This is a common point of confusion. Data Storage is measured in Bytes (e.g., Megabyte/MB) representing file size. Network Speed is measured in bits (e.g., Megabits per second/Mbps). Since there are 8 bits in 1 Byte, a 100 Mbps internet connection actually downloads files at roughly 12.5 MB/s.

How precise are the astronomical distance conversions?

Our matrix uses International System of Units (SI) standards. For instance, calculations involving Light Years and Parsecs utilize scientific notation extending up to 12 decimal points of precision via vanilla JavaScript engines.
